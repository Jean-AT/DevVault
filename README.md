# DevVault

A local-first app where devs dump project ideas and break them into implementable tasks.

## Tech Stack

- **Frontend:** React + TypeScript + Vite
- **Styling:** TailwindCSS v4
- **State:** Zustand with localStorage persistence
- **Routing:** React Router v7
- **Markdown:** react-markdown + remark-gfm
- **Icons:** Lucide React

## Features

- Project idea bank with cards view
- Task breakdown per project with status tracking
- Tags, priorities, and status filters
- Full-text search
- Markdown support for descriptions
- Dark mode
- Export / Import data as JSON
- Local-first — no account needed
- Responsive — works on mobile browsers

## Getting Started

```bash
npm install
npm run dev
```
